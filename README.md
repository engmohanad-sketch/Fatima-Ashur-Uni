# بوابة الرافدين - المنظومة الوطنية للتشخيص التنبؤي
## مشروع مشارك في الملتقى الطلابي السنوي الأول لمشاريع الذكاء الاصطناعي

**إعداد وتطوير:** فاطمة مهند صلاح (طالبة في كلية الطب - جامعة آشور)  
**الشركة الحاضنة:** رواد الجودة للاستشارات الهندسية والمشاريع المستدامة  

---

### 💡 شعار المشروع:
> "الخوارزمية التي تحمي صحة العراقيين، هي ذاتها التي تحرس مياهنا وتصون ثرواتنا".

### 📄 الملخص (Abstract):
تطرح **"بوابة الرافدين"** مقاربة مبتكرة في توظيف الذكاء الاصطناعي الوطني، تتجاوز حدود التطبيقات المنعزلة لتؤسس لمفهوم **"التشخيص العابر للتخصصات"**. يرتكز المشروع على فرضية علمية رائدة تقضي بأن الأنماط البصرية والطيفية المعقدة -سواء في الأنسجة البشرية (كصور الرنين المغناطيسي) أو في تضاريس الأرض- تخضع لمنطق رياضي موحد.

### ⚙️ التمكن التقني (Tech Stack):
توظف المنصة محركاً تشخيصياً ذكياً (AI Diagnostic Engine) يعتمد على:
- **لغة البرمجة:** `Python`
- **البيئة السحابية والبيانات الضخمة:** `Google Earth Engine API` لمعالجة صور الأقمار الصناعية (Sentinel & Landsat)
- **نماذج الرؤية الحاسوبية (Computer Vision):** - Object Detection (للكشف عن الأجسام والآفات)
  - Semantic Segmentation (لتقسيم المساحات وبقع التلوث)
  - Change Detection (لمراقبة التغييرات الزمنية)

### 🎯 المحاور التطبيقية (National Impact):
يقدم المشروع حلولاً استباقية (تشخيص تنبؤي) في 5 قطاعات استراتيجية:
1. **الأمن الغذائي والزراعي:** رصد الإجهاد الخفي للمحاصيل (NDVI).
2. **أمن الطاقة:** كشف الشذوذ الحراري وتسرب الأنابيب النفطية.
3. **المسح الجيولوجي والمائي:** التنقيب واستكشاف الموارد الخفية.
4. **الثروة السمكية:** مراقبة جودة المسطحات المائية.
5. **الصحة العامة والبيئة:** الربط المكاني بين بؤر التلوث البيئي والانتشار الوبائي.
